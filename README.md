# Парсер-менеджер комментариев

Лёгкая утилита на Python для извлечения метаданных статей и текстов комментариев с веб-страниц.

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

---

## 📌 Обзор

`ags_proj` — это минимальный, но расширяемый инструмент для:
- Загрузки веб-страницы по HTTP,
- Извлечения заголовка статьи и текстов комментариев,
- Сохранения результатов в формате JSON,
- Использования в формате библиотеки: импорт функций для использования в собственных скриптах.

Изначально ориентирован на **комментарии Хабра**, но архитектура позволяет легко адаптировать его под другие сайты.

⚠️ Это *прототип / ранняя версия*. Проект не готов к продакшену, но спроектирован с учётом будущего масштабирования.

## 📁 Структура проекта
```
ags_proj/
├── main.py # Точка входа: загрузка → парсинг → сохранение
├── detector.py # Модуль детекции AI
├── config.json             # URL
├── pyproject.toml # Зависимости
├── documentation/ # Документация: архитектура, заметки
├── tests/ # Папка тестирования
├── .gitignore
└── README.md
```

---

## 🔧 Требования

- Python 3.10+
- pip для установки зависимостей

---

## 🎯 Примеры использования

Базовый запуск:
python main.py

С своей конфигурацией:
1. Отредактируйте config.json
2. Запустите python main.py
3. Результаты в output.json
